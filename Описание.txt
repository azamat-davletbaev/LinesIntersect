1. Задание реализовано на язке C#, ASP NET MVC, .NET 4.7.2
2. Программа считывает из файла данные по координатам полигона и линий,
затем проверяет по алгоритму из линейной алгебры на пересечение отрезков.
3. Для простоты не стал реализовывать функцию выбора файла из веб приложения, для запуска необходимо разместить папку проекта 
в C:\Temp
так чтобы файл с данными распологался 
C:\Temp\LinesIntersect\data.txt
4. Восстановите пакеты командой в диспетчере пакетов:
Update-Package Microsoft.CodeDom.Providers.DotNetCompilerPlatform -r
5. В настоящем проекте использовал бы библиотеу OpenLayers в веб страничке, средства БГД OracleSDE(SDO) или PosGIS для проверки пересечения двух геометрических объектов в SQL.
